# Email Generator

![PHP Version](https://img.shields.io/badge/php-%3E%3D8.1-blue)
![License](https://img.shields.io/badge/license-MIT-green)

Email Generator je PHP knihovna pro generování e-mailů pomocí šablonovacího systému [Twig](https://twig.symfony.com/). Tato knihovna umožňuje snadnou práci se šablonami a poskytuje flexibilní způsob, jak integrovat generování e-mailů do vašeho projektu.

---

## Požadavky

- PHP 8.1 nebo novější
- Composer

---

## Instalace

Pro instalaci knihovny použijte Composer:

```bash
composer require lemonade/email-generator
```

---

## Použití

Ukázky použití pro jednotlivé scénáře naleznete v adresáři [demo/](./demo).

---

## Složky a struktura

- `src/` - Obsahuje hlavní zdrojové soubory knihovny.
- `tests/` - Obsahuje jednotkové testy pro knihovnu.
- `demo/` - Obsahuje ukázkové příklady a implementace pro různé scénáře použití.

---

## Testování

Pro spuštění testů použijte příkaz:

```bash
composer test
```

Pokud jste přidali PHPUnit do svých závislostí, budou testy spuštěny pomocí konfigurace `phpunit.xml`.
